EcoCI Downloads & Install

Phase 1.1 install matrix for CLI and VS Code extension adoption.

Install matrix

Surface Install Notes
CLI (recommended) pipx install ecoci Best for isolated global install.
CLI (pip) pip install ecoci Use virtualenv/venv in projects.
CLI (from source) pip install -e . For contributors and local development.
VS Code extension (MVP) Load folder vscode-extensions/ecoci-vscode in VS Code Extension Development Host Marketplace packaging planned for later phase.

Quick verification

  • ecoci version
  • ecoci doctor --provider github
  • ecoci analyze --provider github --json
  • ecoci pr create --provider github --dry-run

For GitHub PR creation, set GITHUB_TOKEN.
