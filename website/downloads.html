Install matrix

Surface Install Notes
CLI (recommended) pipx install ecoci Best for isolated global install.
CLI (pip) pip install ecoci Use virtualenv/venv in projects.
CLI (from source) pip install -e . For contributors and local development.
VS Code extension (MVP) Install generated VSIX from Releases (or load local folder vscode-extensions/ecoci-vscode) Includes Activity Bar dashboard for Analyze/Optimize/PR dry-run.

Quick verification

ecoci version
ecoci doctor --provider github
ecoci analyze --provider github --json
ecoci optimize --provider github --deterministic-patch --patch-file ecoci.patch
ecoci pr create --provider github --dry-run

For write operations, set GITHUB_TOKEN.

Extension verification

  • Open Activity Bar → EcoCI
  • Run Analyze to fetch findings and quality gate
  • Run Optimize to inspect change plan
  • Run PR Dry Run to preview PR payload safely

Release artifacts

Each tagged release publishes artifacts for direct download.

  • Python wheel: ecoci-*.whl
  • Source package: ecoci-*.tar.gz
  • VS Code extension: ecoci-vscode.vsix
