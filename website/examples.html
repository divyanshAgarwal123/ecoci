EcoCI Command Examples

Copy-paste recipes for common CI optimization workflows.

Home Quickstart Repository

Analyze with report output

ecoci analyze --provider github --run-id <actions_run_id> --markdown ecoci-report.md

Generate deterministic patch

ecoci optimize --provider github --deterministic-patch --patch-file ecoci.patch

Inspect optimization diff

ecoci optimize --provider github --show-diff

Safe PR preview

ecoci pr create --provider github --dry-run --json

Create optimization PR + dashboard

ecoci pr create --provider github --run-id <actions_run_id> --title "EcoCI: Optimize workflow"

Setup diagnostics

ecoci doctor --provider github --json

Troubleshooting

  • If repo is not auto-detected, pass --repo owner/repo.
  • If PR creation fails, verify GITHUB_TOKEN permissions.
  • If run metrics are missing, include a valid --run-id.
