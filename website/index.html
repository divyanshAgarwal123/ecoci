Optimize CI workflows for speed, cost, and carbon

EcoCI analyzes GitHub Actions workflows, identifies risky and expensive patterns, proposes safe fixes with confidence scoring, and helps automate PR-based optimization.

Fast diagnosis

Run doctor and analyze to get security findings, quality gates, and prioritized recommendations.

Safe optimization

Generate deterministic patches with unified diff previews and confidence-scored fixes before applying.

PR automation

Create optimization PRs with rollback guidance and KPI projection blocks for reviewer confidence.

Quick start

ecoci doctor --provider github
ecoci analyze --provider github --json
ecoci optimize --provider github --deterministic-patch --patch-file ecoci.patch
ecoci pr create --provider github --dry-run
