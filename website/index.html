EcoCI

Provider-pluggable CI optimization toolkit for GitHub-first workflows.

Install CLI

Use Python package install:

pip install ecoci or pipx install ecoci

Required for GitHub PR actions: GITHUB_TOKEN

See full install matrix →

Quick start

  • ecoci doctor --provider github
  • ecoci analyze --provider github --json
  • ecoci optimize --provider github --json
  • ecoci pr create --provider github --dry-run

VS Code extension

MVP extension scaffold is in this repository:

vscode-extensions/ecoci-vscode

Commands included:

  • EcoCI: Analyze Current Repo
  • EcoCI: Optimize CI Workflows
  • EcoCI: Create Optimization PR
  • EcoCI: Show Carbon & Cost Summary
