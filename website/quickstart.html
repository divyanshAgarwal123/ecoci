EcoCI Quickstart

Install, verify, analyze, optimize, and create a safe PR in under 5 minutes.

Home Downloads Repository

1) Install

pipx install ecoci
# or
pip install ecoci

2) Set GitHub token

Required for write operations like creating PRs.

export GITHUB_TOKEN=<your_token>

3) Verify setup

ecoci version
ecoci doctor --provider github

4) Analyze and optimize

ecoci analyze --provider github --json
ecoci optimize --provider github --show-diff
ecoci optimize --provider github --deterministic-patch --patch-file ecoci.patch

5) Create optimization PR safely

ecoci pr create --provider github --dry-run --json
# when ready
ecoci pr create --provider github --title "EcoCI: Optimize workflow"

VS Code dashboard flow

  • Open Activity Bar → EcoCI
  • Click Analyze to load findings and quality gate
  • Click Optimize to preview changes
  • Click PR Dry Run to preview PR payload
